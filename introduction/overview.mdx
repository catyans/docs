## What is VINE?

VINE is a **self-trading (proprietary) BTC fund** that uses AI agents to generate trading signals, combined with rigorous quantitative risk management. The platform exposes a comprehensive API for:

- **Investors** — Subscribe, monitor NAV, view on-chain records, download reports
- **Integrators** — Feed external signals via webhooks, build custom dashboards
- **Auditors** — Verify on-chain transactions, access stress test results

## Key Features

| Capability | Details |
| --- | --- |
| **AI Signal Generation** | Claude LLM agents analyze news, macro data, and market microstructure in real-time |
| **Multi-Strategy** | News event, macro regime, funding rate, and webhook signals combined via decay-weighted ensemble |
| **4-Layer Risk Management** | Agent → Supervisor → Hard Limits → Circuit Breaker hierarchical protection |
| **On-Chain Transparency** | All subscriptions and redemptions recorded on-chain with verifiable proofs |
| **Real-Time Monitoring** | Live NAV, drawdown, leverage, and risk metrics via API |
| **Institutional Reports** | Daily, weekly, monthly PDF reports with full TCA analysis |

## How It Works

<Steps>
  <Step title="Signal Generation">
    AI agents (news analysis, macro regime, funding rate) generate directional signals with conviction scores and decay half-lives.
  </Step>
  <Step title="Signal Combination">
    Signals are combined via weighted ensemble: `w = Σ(α_agent × conviction × direction × decay_factor)`
  </Step>
  <Step title="Portfolio Construction">
    Volatility targeting, regime overlay, and leverage constraints produce a target BTC position.
  </Step>
  <Step title="Execution & Risk">
    Orders pass through 4-layer risk checks before execution. TCA monitors slippage and implementation shortfall.
  </Step>
</Steps>
