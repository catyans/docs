# API Key Authentication

API Keys are used for **server-to-server integrations** and **webhook ingestion**. Each key pair consists of an API Key (public identifier) and an API Secret (for signing requests).

## Creating API Keys

1. Navigate to **Dashboard → Settings → API Keys**
2. Click **Generate New Key**
3. Select permissions scope: `read`, `trade`, or `admin`
4. Copy and store the API Secret securely

<Warning>
  The API Secret is only displayed once. If lost, you must rotate to a new key pair.
</Warning>

## Request Signing

All authenticated requests must include three headers:

| Header | Description |
| --- | --- |
| `X-API-Key` | Your public API key |
| `X-Signature` | HMAC-SHA256 signature of the request |
| `X-Timestamp` | Unix timestamp (seconds). Requests older than 30s are rejected. |

### Signature Construction

```
signature = HMAC-SHA256(
  key = api_secret,
  message = "{timestamp}.{method}.{path}.{body}"
)
```

### Python Example

```python
import hmac
import hashlib
import time
import json
import requests

API_KEY = "pk_live_abc123"
API_SECRET = "sk_live_xyz789"
BASE_URL = "https://api.poacapital.io/v1"

def signed_request(method: str, path: str, body: dict = None):
    timestamp = str(int(time.time()))
    body_str = json.dumps(body, separators=(',', ':')) if body else ""

    message = f"{timestamp}.{method.upper()}.{path}.{body_str}"
    signature = hmac.new(
        API_SECRET.encode(),
        message.encode(),
        hashlib.sha256
    ).hexdigest()

    headers = {
        "X-API-Key": API_KEY,
        "X-Signature": signature,
        "X-Timestamp": timestamp,
        "Content-Type": "application/json",
    }

    resp = requests.request(
        method, f"{BASE_URL}{path}",
        headers=headers,
        json=body
    )
    return resp.json()

# Example: Get portfolio
portfolio = signed_request("GET", "/portfolio")
```

### TypeScript Example

```typescript
import crypto from 'crypto';

const API_KEY = 'pk_live_abc123';
const API_SECRET = 'sk_live_xyz789';

function signRequest(method: string, path: string, body?: object): Record<string, string> {
  const timestamp = Math.floor(Date.now() / 1000).toString();
  const bodyStr = body ? JSON.stringify(body) : '';
  const message = `${timestamp}.${method.toUpperCase()}.${path}.${bodyStr}`;

  const signature = crypto
    .createHmac('sha256', API_SECRET)
    .update(message)
    .digest('hex');

  return {
    'X-API-Key': API_KEY,
    'X-Signature': signature,
    'X-Timestamp': timestamp,
  };
}
```

## Key Permissions

| Scope | Access |
| --- | --- |
| `read` | Portfolio, NAV, risk status, reports |
| `trade` | All `read` + subscription, redemption |
| `admin` | All `trade` + circuit breaker reset, config changes |

## Key Rotation

```bash
# Rotate API key (invalidates old key after 24h grace period)
curl -X POST https://api.poacapital.io/v1/auth/keys/rotate \
  -H "X-API-Key: pk_live_abc123" \
  -H "X-Signature: ..." \
  -H "X-Timestamp: ..."
```

<Note>
  Old keys remain valid for 24 hours after rotation to allow seamless migration.
</Note>
