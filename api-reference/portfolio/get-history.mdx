# Get NAV History

Retrieve historical NAV snapshots for the fund. Useful for building equity curves, drawdown charts, and performance dashboards.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. Requires `investor`, `viewer`, or `pm` role.
</ParamField>

## Query Parameters

<ParamField query="period" type="string" default="30d">
  Lookback period. Options: `7d`, `30d`, `90d`, `1y`, `all`
</ParamField>

<ParamField query="interval" type="string" default="daily">
  Data granularity. Options: `hourly`, `daily`, `weekly`
</ParamField>

## Request

```bash
curl "https://api.yourdomain.com/v1/portfolio/history?period=30d&interval=daily" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..."
```

## Series Fields

| Field | Type | Description |
| --- | --- | --- |
| `timestamp` | string | ISO 8601 snapshot time |
| `nav` | number | Net Asset Value at snapshot |
| `daily_pnl` | number | Day-over-day P&L in USDT |
| `drawdown` | number | Drawdown from HWM (negative) |
| `leverage` | number | Portfolio leverage at snapshot |
