# Get Performance Metrics

Retrieve comprehensive performance analytics including Sharpe ratio, Sortino ratio, maximum drawdown, win rate, and alpha/beta decomposition.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. Requires `investor`, `viewer`, or `pm` role.
</ParamField>

## Query Parameters

<ParamField query="period" type="string" default="90d">
  Analysis period. Options: `30d`, `90d`, `180d`, `1y`, `all`
</ParamField>

## Request

```bash
curl "https://api.yourdomain.com/v1/performance?period=90d" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..."
```

## Metrics Reference

| Metric | Formula | Description |
| --- | --- | --- |
| `sharpe_ratio` | (R - Rf) / σ | Risk-adjusted return (annualized) |
| `sortino_ratio` | (R - Rf) / σ_down | Downside risk-adjusted return |
| `calmar_ratio` | R_annual / MaxDD | Return per unit of max drawdown |
| `profit_factor` | Gross Profit / Gross Loss | Profitability ratio |
| `win_rate` | Winning Trades / Total Trades | Percentage of profitable trades |
| `beta_to_btc` | Cov(R, R_btc) / Var(R_btc) | Systematic BTC exposure |
| `alpha_annualized` | R - β × R_btc | Excess return vs BTC buy-and-hold |
