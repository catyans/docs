# Get On-Chain Proof

Retrieve verifiable on-chain proof for a specific transaction. Used for audit and compliance verification.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. Requires `investor` role.
</ParamField>

## Path Parameters

<ParamField path="tx_hash" type="string" required>
  Transaction hash (0x-prefixed hex string)
</ParamField>

## Request

```bash
curl https://api.yourdomain.com/v1/onchain/proof/0xabc123def456... \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..."
```
