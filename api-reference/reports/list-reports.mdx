# List Reports

Retrieve a list of available reports for download.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. Requires `investor`, `viewer`, or `pm` role.
</ParamField>

## Query Parameters

<ParamField query="type" type="string" default="all">
  Report type filter: `daily`, `weekly`, `monthly`, `all`
</ParamField>

## Request

```bash
curl "https://api.yourdomain.com/v1/reports/list?type=monthly" \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..."
```
