# Get Risk Status

Retrieve the current state of the 4-layer risk management system, including drawdown protocol level, circuit breaker status, and value-at-risk metrics.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. Requires `viewer` or `pm` role.
</ParamField>

## Request

```bash
curl https://api.poacapital.io/v1/risk \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..."
```

## Drawdown Protocol Levels

| Level | Trigger | Risk Multiplier | Action |
| --- | --- | --- | --- |
| **0** | DD > -5% | 1.00 | Normal operations |
| **1** | DD ≤ -5% | 0.75 | Position sizes reduced 25% |
| **2** | DD ≤ -10% | 0.50 | Reduced 50%, short-vol disabled |
| **3** | DD ≤ -15% | 0.00 | All positions flattened, PM approval required |

## Circuit Breaker Triggers

The circuit breaker halts all trading when any of these conditions are met:

- Drawdown exceeds **-15%**
- More than **5 consecutive** API errors
- Price moves **>10%** in a single 1-minute candle

<Warning>
  When the circuit breaker is active, all order submission is blocked. Only a PM can reset it via `POST /risk/circuit-breaker/reset`.
</Warning>
