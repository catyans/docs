# Run Stress Test

Execute 9 stress test scenarios (5 historical + 4 hypothetical) against the current portfolio and return projected P&L impacts.

## Authentication

No authentication required. This is a public endpoint.

## Request

```bash
curl https://api.poacapital.io/v1/stress
```
