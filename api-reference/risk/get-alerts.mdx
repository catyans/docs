# Get Alerts

Retrieve the most recent system alerts including risk events, data quality issues, and circuit breaker activations.

## Authentication

No authentication required.

## Request

```bash
curl https://api.yourdomain.com/v1/alerts
```

## Response Example

```json
{
  "alerts": [
    {
      "id": "alt_001",
      "level": "WARNING",
      "category": "risk",
      "message": "Drawdown level changed to 1 (-5.2%)",
      "timestamp": "2026-03-09T10:30:00Z",
      "details": {
        "drawdown_pct": -0.052,
        "risk_multiplier": 0.75
      }
    },
    {
      "id": "alt_002",
      "level": "INFO",
      "category": "data_quality",
      "message": "Price data gap detected: 3 missing 1-min bars",
      "timestamp": "2026-03-09T08:15:00Z",
      "details": {
        "symbol": "BTC/USDT",
        "missing_bars": 3,
        "gap_start": "2026-03-09T08:12:00Z"
      }
    }
  ]
}
```

## Alert Levels

| Level | Description | Notification Channels |
| --- | --- | --- |
| `CRITICAL` | Circuit breaker, system failure | Telegram + Discord + Email |
| `WARNING` | Drawdown change, risk threshold | Telegram + Discord |
| `INFO` | Data quality, routine events | Discord only |
